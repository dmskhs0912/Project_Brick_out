1. 프로그램 실행 환경 
 - Visual Studio 2022 Debug x64
 - Intel i7-7700k
 - 16GB RAM
 - Windows 10 Education

2. 실행 방법
 2-1. 프로젝트 폴더에 bin/Brick_out_v02_debug.exe로 프로그램을 실행할 수 있다.
 2-2. Start Game 버튼으로 게임 시작, View Ranks 버튼으로 순위 열람, Exit 버튼으로 게임 종료를 할 수 있다.
 2-3. 게임 시작 시 자동으로 Stage 1부터 시작하며 우클릭으로 공을 발사해 게임을 시작할 수 있다. 공이 패들에 튕기면 -20점, 벽돌에 튕기면 +50점,
게임 오버 시 -300점, 스테이지 클리어 시 +300점이 주어진다. 게임 오버 또는 스테이지 클리어 시 3글자의 이름을 입력할 수 있다. 이 정보는 점수와 함께
랭크화 되어 bin/data/ 폴더에 텍스트 파일로 저장된다. 다음 게임 실행 시 해당 랭크 정보를 자동으로 불러온다.
 2-4. View Ranks 버튼으로 랭크 열람 시 'A'로 이전 페이지(스테이지), 'D'로 다음 페이지(스테이지)의 랭크 정보를 열람할 수 있고, 'Q'로 시작 메뉴로 돌아 갈 수 있다.
 2-5. Bricks.json 파일을 편집해 스테이지 벽돌 구성을 변경할 수 있다. 

3. 기타 정보
 3-1. 빨간 벽돌은 3번만에 부숴지고, 초록 벽돌은 2번, 파란 벽돌은 1번에 부숴진다.
 3-2. 공을 패들의 중앙에 맞출 수록 위 방향으로 튕겨 나가고 오른쪽/왼쪽 끝에 맞출 수록 오른쪽/왼쪽 방향으로 튕겨 나간다.
 3-3. 고득점을 위해서는 패들과의 한 번의 튕김으로 여러 번의 벽돌과의 충돌을 만들어야 한다.
